# Privacy Policy for Bookshelf

**Effective Date:** February 19, 2026  
**Last Updated:** March 30, 2026

Huda AbuSharr (independent developer) (“we,” “our,” or “us”) operates the Bookshelf mobile application.

This Privacy Policy explains what information we collect, how we use it, and your choices.

## 1. Information We Collect

We may collect the following categories of information:

- **Account Information:** email address, user ID, and display name (for signed-in users).
- **User Library Data:** favorite/read/saved book states and sync metadata.
- **Search and Discovery Data:** in-app search terms and movie-to-book query inputs.
- **Book Scanner Data:** images selected from gallery or captured with camera.
- **AI Feature Request Data:** prompts and related metadata.
- **Purchase Data:** premium status and purchase metadata.
- **Device and Diagnostics Data:** app version, device model, OS version, crash logs.
- **Messaging Data:** Firebase Cloud Messaging (FCM) token.
- **Advertising Data:** ad-related identifiers and interaction events.
- **Content Request Data:** request parameters used to fetch book metadata.

## 2. How We Use Information

We use collected data to:

- authenticate users and manage accounts,
- sync reading state across devices,
- verify premium purchases,
- provide app features,
- generate AI-powered summaries,
- improve performance and reliability,
- send optional notifications,
- display ads.

## 3. Third-Party Services

Bookshelf uses third-party services, including:

- Firebase
- Google AdMob
- Google Play Billing
- Google Books API
- Google Gemini API

These providers process data under their own policies.

## 4. Data Sharing

We do not sell personal data.

We may share data only when required for services, legal reasons, or protection.

## 5. Data Retention

We retain data only as long as necessary for service operation and legal compliance.

## 6. Security

We use reasonable safeguards, but no system is 100% secure.

## 7. Children’s Privacy

Bookshelf is intended for adults (18+).
